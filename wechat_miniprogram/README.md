# 日历记录小助手 - 微信小程序

这是一个配合日历网站使用的微信小程序，支持语音交互和蓝牙连接。

## 功能特性

1. **语音助手** - 通过语音或文字与AI对话，记录月经、学习、工作日程
2. **蓝牙连接** - 连接蓝牙耳机/挂件设备
3. **记录查看** - 查看和管理所有日历记录

## 项目结构

```
wechat_miniprogram/

├── app.json            # 小程序配置

```

## 使用前准备

### 1. 日历网站配置

确保日历网站已启动并运行：

```bash
cd calendar_app
source venv/bin/activate
python app.py
```

### 2. 修改API地址

在 `utils/api.js` 中修改 `apiBase` 为你的服务器地址：

```javascript
apiBase: 'http://你的服务器IP:5000'
```

### 3. 安装依赖

小程序需要使用微信的语音插件：
- 搜索并添加 "微信同声传译" 插件
- 在 `app.json` 中配置插件

## 使用方法

### 1. 连接蓝牙设备

1. 打开小程序，进入 "蓝牙设备" 页面
2. 点击 "搜索设备"
3. 选择你的蓝牙耳机/挂件设备
4. 连接成功后返回

### 2. 使用语音助手

1. 进入 "语音助手" 页面
2. 按住中间的麦克风按钮说话
3. 松开按钮，AI会自动回复

### 3. 快捷指令

- "帮我记录月经" - 记录月经
- "查看本月记录" - 查看记录
- "帮我记录学习" - 记录学习
- "帮我记录工作" - 记录工作

## 意图识别示例

小程序会将用户语音发送给AI，AI会理解意图并回复。目前支持：

| 用户说 | AI回复 |
|--------|--------|
| 帮我记录月经 | 好的，请告诉我开始日期 |
| 明天9点开会 | 已记录，明天9点有会议安排 |
| 查看今天的记录 | 今天暂无记录 |

## API接口

### 语音对话

```
POST /api/voice/chat
Content-Type: application/json

{
  "message": "用户说的话"
}

Response:
{
  "success": true,
  "reply": "AI回复"
}
```

## 注意事项

1. 微信小程序需要在微信开发者工具中打开
2. 蓝牙功能需要真机调试
3. 语音识别需要使用微信同声传译插件
4. 确保手机和服务器在同一网络下

## 进阶开发

### 添加意图识别

可以在后端 `app.py` 中扩展 `/api/voice/chat` 接口，添加对常见意图的自动处理：

```python
# 意图识别示例
if "月经" in message or "经期" in message:
    # 处理月经记录
    return handle_period_record(message)
elif "学习" in message:
    # 处理学习记录
    return handle_study_record(message)
```

### 自定义语音

可以修改 `utils/api.js` 中的 `textToSpeech` 函数使用其他TTS服务。

## 故障排除

1. **无法连接蓝牙**
   - 检查手机蓝牙是否打开
   - 确保设备已开机

2. **API调用失败**
   - 检查服务器是否运行
   - 检查网络连接
   - 确认API地址配置正确

3. **语音识别失败**
   - 检查是否安装语音插件
   - 确认录音权限已开启
